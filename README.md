# Home Cleaning UI

Home cleaning Ui in Flutter


## Getting Started

#### 1. Copy or Download code in your system 

```
git clone https://github.com/om-chauhan/Home-Cleaning-UI-in-Flutter.git
```

#### 2. Got To Directry
```
cd Home-Cleaning-UI-in-Flutter
```
#### 3. Run Flutter App

```
flutter run
```
